# RC Car with Robotic Arm

A remote-controlled car with an integrated robotic arm built using Arduino and C++.

## Features
- Wireless remote control
- Robotic arm for pick-and-place tasks
- Bluetooth Module (HC-05)

## Components
- Arduino UNO
- Motor Driver (L298N)
- Servo Motors
- RC Transmitter & Receiver
- Bluetooth Module (HC-05)
- Step Up Module (Boost Converter) – 10V output

## Tools
- Arduino IDE
- C++
